Загрузка на GitHub:
1. Открой репозиторий gastroconnect.
2. Add file -> Upload files.
3. Перетащи ВСЕ файлы и папку assets из этого архива.
4. Commit changes.
5. Через 1-3 минуты открой https://gastroconnect.ru и нажми Ctrl+F5.

admin.html — временная локальная панель управления. Заявки видны только в том браузере, где их отправили. Для настоящей базы нужен Supabase.
